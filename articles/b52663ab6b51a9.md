# 雑多な例たち
## 乗算器号の省略
```julia
julia> e1, f1, g1 = 2,3,4
(2, 3, 4)

julia> 5e1
50.0

julia> 5f1
50.0f0

julia> 5g1
20
```

これは..のためですね。
下付き文字を使えば解決できます：

```julia
julia> e₁, f₁, g₁ = 2,3,4
(2, 3, 4)

julia> 5e₁
10

julia> 5f₁
15

julia> 5g₁
20
```

## falseでの掛け算

```julia
julia> NaN * 0
NaN

julia> Inf * 0
NaN

julia> NaN * false
0.0

julia> Inf * false
0.0
```

## 推移的でない等号

```julia
julia> using LinearAlgebra

julia> I(3)
3×3 Diagonal{Bool, Vector{Bool}}:
 1  ⋅  ⋅
 ⋅  1  ⋅
 ⋅  ⋅  1

julia> I(3) == I == I(4)
true

julia> I(3) == I(4)
false
```
